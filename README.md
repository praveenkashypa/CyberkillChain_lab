# CyberkillChain_lab
This repository supports a TryHackMe lab titled Cyber Kill Chain. Included in the repo is the lab PDF and space for an ordered notes and artifacts you gather while solving the lab. The included PDF (original lab material) is stored here for reference.
Cyber_Kill-Chain
The content is intended for learning and defensive/offensive security training only. Do not reuse any exploit instructions against systems you do not own or have explicit permission to test.

Objectives & learning outcomes

After completing the lab and using materials in this repo you should be able to:

Understand the stages of the Cyber Kill Chain and map typical attacker activities to each stage.

Perform reconnaissance, scanning, and basic enumeration of a target VM.

Identify and exploit simple misconfigurations (where applicable to the lab).

Document findings clearly and produce a professional lab write-up.

Improve familiarity with common pentesting tools and methodology.
